Características de la Aplicación
Funcionalidades Principales:

Tres Métodos Numéricos Implementados:
  Bisección
  Newton-Raphson
  Secante

Cuatro Problemas Predefinidos (cargables con un clic):
  Problema 1: x³ - e^(0.8x) = 20
  Problema 2: 3sin(0.5x) - 0.5x + 2 = 0
  Problema 3: x³ - x²e^(-0.5x) - 3x = -1
  Problema 4: cos²x - 0.5xe^(0.3x) + 5 = 0

Visualización Gráfica:
  Gráfica de la función
  Visualización de la raíz encontrada
 
Tabla de Iteraciones Completa:
  Muestra todos los valores calculados en cada iteración
  Exportable a CSV para análisis en Excel

Parámetros Configurables:
  Tolerancia
  Número máximo de iteraciones
  Intervalos iniciales
